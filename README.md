# React Shopping Cart

A simple shopping cart application built with React as part of the *Developing Front-End Apps with React* course.

## Overview

This project demonstrates core React concepts including component-based architecture, state management, and deploying a React app using the `gh-pages` package. The app allows users to browse products, add items to the cart, and view their selections.

## Features

- Browse a list of products with images and descriptions
- Add and remove items from the shopping cart
- View cart summary with total price
- Responsive and user-friendly UI
- Deployment to GitHub Pages using `gh-pages`

## Technologies Used

- React
- JavaScript (ES6+)
- CSS Modules / Styled Components (depending on your styling choice)
- `gh-pages` for deployment
- Cloud IDE for development environment
